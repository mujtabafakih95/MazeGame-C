# 🧩 Maze Runner Game (C Language)

A console-based Maze Runner game developed in C featuring player movement, AI enemy, scoring system, and win/lose conditions.

 🎮 Features
- Player movement using WASD keys
- AI enemy that chases player
- Score tracking system
- Win/Lose conditions
- Console-based gameplay
 How to Run

  bash
gcc maze_runner.c -o game
./game
