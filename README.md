🧩 Maze Runner Game (C Language)

A console-based Maze Runner game built using C, featuring player navigation, AI-driven enemy movement, scoring system, and win/lose mechanics. This project demonstrates fundamental programming concepts including loops, conditionals, arrays, and basic AI logic.

⸻

🚀 Features
	•	🎮 Player movement using WASD controls
	•	🤖 AI enemy that dynamically chases the player
	•	🧠 Basic path-following enemy logic
	•	💰 Score tracking system
	•	🏁 Win and lose conditions
	•	🧱 Maze-based obstacle navigation
	•	💻 Lightweight console-based gameplay

🛠️ Tech Stack
	•	Language: C Programming
	•	Concepts Used: Arrays, Loops, Conditionals, Basic AI Logic
	•	Platform: Console Application

🎯 Game Objective

Navigate through the maze, avoid the enemy, and reach the exit point before getting caught.
🧠 Project Highlights
	•	Implemented simple AI behavior for enemy movement
	•	Designed maze-based navigation system
	•	Real-time console updates
	•	Beginner-friendly game structure in C
